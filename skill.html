Skill Saya

Selama masa sekolah, saya telah mendalami berbagai aspek pengembangan web yang membuatnya tidak hanya berfungsi, tetapi juga nyaman digunakan di berbagai perangkat. Saya memulai dengan membangun fondasi yang kokoh, di mana setiap bagian halaman memiliki tujuan yang jelas, membuatnya mudah dipahami oleh pengguna maupun mesin pencari. Kemudian, saya mengasah kemampuan untuk memastikan website terlihat rapi dan bekerja sempurna, baik saat diakses dari layar komputer yang besar maupun dari genggaman ponsel. Puncaknya, saya dapat menghidupkan website dengan menambahkan interaksi yang dinamis, seperti membuat timer yang bisa diatur atau membuat elemen di halaman bergerak sesuai dengan keinginan pengguna, menjadikan website tidak hanya informatif, tetapi juga interaktif dan menarik. Berikut adalah beberapa keahlian yang saya dapatkan selama belajar bahasa pemrograman:

2. Menguasai Elemen dan Atribut Dasar

3. Mampu Membuat Tabel dan Formulir

1. Menguasai styling modern termasuk flexbox, grid, dan animasi.

2. Mampu Membuat Struktur Website dengan Semantic HTML

3. Mampu Menguasai Animasi dan Transisi CSS

JavaScript

1. Dapat membuat interaksi dinamis dengan DOM dan logika dasar.
